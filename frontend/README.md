# 乒器通 Web 原型

入口文件：`index.html`

## 已实现

- 响应式布局：电脑端左侧导航工作台，手机端底部标签栏。
- 选型推荐：预算、水平、年龄、训练频率、打法、痛点输入后生成底板/正手/反手方案。
- 低预算分流：预算低于自选底板 + 两面胶皮的可行区间时，自动推荐成品拍过渡方案，避免 300 元预算误推 800 元以上专业配置。
- 个性化选配：支持自动全套推荐、定好底板选择两面胶皮、单独定制正手胶皮、单独定制反手胶皮四种模式。
- 正反手胶皮参数：正手支持红双喜狂飙系列细分、蝴蝶、亚萨卡、尼塔库、多尼克、斯帝卡、挺拔、骄猛等品牌路线；反手支持按品牌和硬度/弹性路线选择。
- 推荐结果：动态展示对应底板实物图、正反手胶皮包装图和渠道采集价快照；缺少授权图的分支只显示待补图状态，不再使用模拟包装。
- 图源覆盖：当前已接入 Butterfly、DHS Canada、STIGA、Yasaka、Nittaku、VICTAS、DONIC、TIBHAR、XIOM、ANDRO、JOOLA 等官方/公开零售产品图；仍有 Hugo HAL、STIGA Pro Carbon、Palio Expert 3、Killerspin JET800 4 个条目保留待补图状态，避免误配图。
- 预算滑杆：整拍预算上限设为 3000 元，覆盖主流顶配底板 + 两面胶皮的参考上限。
- 趣味测试：15 题打法性格测试，结果可带入选型。
- 器材评分：已拆分为底板、胶皮、成品拍三个模块；首批 54 条市场索引支持品牌、型号、别名和球友黑话搜索。
- 底板商家参数：新增品牌、系列、型号、材质/结构、层数、厚度、拍面、反应/震动等结构化字段；首批 8 个条目已按 Butterfly Global 与 STIGA 官方页面核验，未核型号只显示“官网参数待核”；底板模块支持按品牌、材质/结构、层数筛选。
- 别名校准：张继科 ALC 与多尼克 Zhang Jike New Era 分开维护，“红张”等球友口径先标记为待核别名。
- 冠军装备：新增 2004 至今世界冠军名将装备索引，可按球员、国家、头衔、底板、胶皮搜索，并保留 A/B/C 证据等级。
- 冠军照片：冠军卡片支持公开授权赛场照和图源链接，优先使用 Wikimedia Commons 等可溯源图片；缺少可靠图源的球员不强行占位。
- 球友圈：文字发帖、点赞、敏感内容进入人工复核。
- 个人资料：昵称、地区、开球网积分、当前装备，使用浏览器本地存储。
- 数据可信度：A/B/C/UGC 证据分层和首版数据范围。

## 后续接入

- 将 `app.js` 内的本地数组替换为后端 API。
- 将评论、帖子、评分、个人资料接入数据库。
- 将审核状态接入内容安全服务和人工后台。
- 上线时可直接部署到静态托管，也可以接入 Next.js/Nuxt 等框架继续工程化。
- 价格正式上线建议由后端定时抓取淘宝、京东、授权店铺或自营 SKU，清洗异常低价后写入商品价格快照；静态原型只保留当前快照和展示结构。
- 选型页支持 `?budget=300&level=beginner&age=12` 这类测试参数，便于复现具体推荐场景。
